# ZLCommonCell
一些通用的tableview和collectionview的cell，header。


### 普通cell，
    带左文字，左图片，右文字，右图片，可以选择隐藏图片

![](https://github.com/czl0325/ZLCommonCell/blob/master/demo1.png?raw=true)


### header
    带左文字，右文字，右图片
    
    
### footer
    单独一个按钮的footer
    
### ZLLabelCell
    应用于collectionview的标签页面，内含一个label
